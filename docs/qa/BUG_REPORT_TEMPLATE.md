# Шаблон баг-репорта

## Название бага

Коротко: что сломано и где.

## Серьёзность

Выберите одно:

- `blocker` — невозможно пользоваться сценарием;
- `major` — важная функция сломана;
- `minor` — мешает, но можно обойти;
- `cosmetic` — внешний вид, подпись, текст, мелкая UX-проблема.

## Режим

Выберите подходящий пункт:

- Главная страница
- Рисунок электродов
- Электроды функциями
- Нарисовать электроды
- Импорт поля `.npz`
- Импорт потенциала `.npz`
- RF/DC роли
- Псевдопотенциал
- Диаграмма Матье
- Подбор параметров
- Экспорт
- Другое
