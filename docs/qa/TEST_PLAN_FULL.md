# Полный ручной тест-план

Этот чеклист предназначен для ручного тестирования Streamlit-приложения
`charged_particle_trap`. Тестировщик проверяет интерфейс, устойчивость,
понятность ошибок, загрузку/сохранение и базовые сценарии расчёта. Тестировщик
не обязан доказывать физическую корректность модели ловушки.

## Как заполнять

В колонке `Результат` укажите `Pass`, `Fail` или `Blocked`. В колонке
`Заметки/скриншот` укажите имя скриншота, номер баг-репорта или короткий
комментарий.

Приоритеты:

- `P0` — обязательно должно работать;
- `P1` — важная функция;
- `P2` — полезная UX-проверка.
